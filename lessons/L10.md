# LESSON 10: "OUR PROCESS IS FASTER NOW"
**Grammar:** Comparative/Superlative Adverbs | **Theme:** Process Improvement

## VOCABULARY (8 words)
efficiently, quickly, carefully, thoroughly, professionally, effectively, smoothly, successfully

**Audio:** "Recruitment Improvements This Year" - HR Director: "This year we're working more efficiently. We're screening CVs more quickly. We're conducting interviews more professionally. Our time-to-hire is faster than before..."

**Grammar:**
```
REGULAR: more/most + -ly adverb
quickly → more quickly → most quickly
carefully → more carefully → most carefully

IRREGULAR:
well → better → best
fast → faster → fastest
hard → harder → hardest
```

**Before/After Comparisons:**
- We work faster NOW than before
- We screen more carefully NOW than last year
- Our process runs more smoothly NOW
